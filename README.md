# E-Commerce-Backend

## Description
- This is the backend demo of a ecom website. The goal was to demoonstrate data handling and work with the database.
## Installation
- git clone git@github.com:bsellers1/ecom-back-end.git
- npm i
- source ./db/schema.sql
- npm run seed
- nodemon server.js
